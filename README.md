# 🚀 强***教务系统过程性成绩自动录入工具

> **作者：小怪兽** | **版本：V5.5 **

这是一个基于 Python + Selenium + Tkinter 开发自动化填报工具。旨在解决期末老师们录入成绩时某教务系统某些版本不支持导入功能等痛点。

如果您有新的想法和完善需求，可联系发布者共同完善。

## ✨ 核心特性

- **🤖 全自动智能录入**：支持动态匹配 Excel 表头与网页输入框，安全防错。
- **⚡ 极速比例配置**：支持在 Excel 表头直接写入比例（如 `考勤:10`），或通过独立UI批量秒设成绩比例。
- **🛡️ 终极保存机制**：底层注入 JS 原生触发 `saveForm()`，无视前端遮挡，100% 解决保存卡死问题。
- **🌐 跨页与驱动自适应**：自动处理教务系统跨页丢失班级的问题；全自动下载匹配本机的 Chrome/Edge/Firefox 离线驱动。

## 📦 如何使用

1. 本.exe可执行文件可以直接下载双击使用

2. 若需要源代码，可联系作者提供，源代码执行需满足以下环境。

3. 确保电脑已安装 Python 3.8以及上。

4. 安装必要的依赖库：

   ```bash
   pip install pandas openpyxl selenium webdriver-manager
   ```
